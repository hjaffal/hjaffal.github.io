What is the Sproochentest?

The Sproochentest is the Luxembourgish language test used for Luxembourg nationality. It checks speaking at A2 level and listening at B1 level.

About This Guide

Why I built this

I built this guide during my own Sproochentest preparation. After months of language courses, I realized the last stretch before the exam needed something different — not more grammar lessons, but focused, structured practice on exactly what the test asks for: speaking about everyday topics, describing images, and understanding spoken Luxembourgish.

I couldn't find a single resource that brought all of this together in one place. So I made one.

What this is (and isn't)

This is a final-month prep guide. It assumes you've already been learning Luxembourgish — through courses, conversation, or self-study. It does not replace language learning. It's the last push: structured practice questions, vocabulary drills, listening exercises, and image description frameworks designed specifically for the Sproochentest format.

Think of it as exam prep, not a course.

How it was built

This entire platform — the vocabulary engine, the audio generation pipeline, the interactive quizzes, the spaced repetition system — was built in a single session using Kiro, an AI-powered development environment. From requirements to deployed product in one sitting. The vocabulary data was curated manually from my own study notes, then structured and coded with Kiro's help.

Credits & Acknowledgements

This guide would not be possible without:

  • LOD (Lëtzebuerger Online Dictionnaire) — The reference dictionary for Luxembourgish. Essential for verifying spelling and usage.
  • The Reddit and expat communities in Luxembourg who share their exam experiences and practice questions.
This is not an official INLL resource. It is an independent study guide created by a fellow test-taker. Practice questions are curated from community-shared experiences, not from official exam papers.
